# Pixel-Art-Maker
A simple pixel art tool built with HTML, CSS, and JavaScript.

## Features
* **Drawing Tools:** Pen, Eraser, and Flood Fill (Bucket).
* **Grid Sizes:** Switch between 16x16, 32x32, and 64x64.
* **Zoom:** Zoom in and out of the canvas.
* **Color Picker:** Use preset colors or a custom color selector.
* **Export:** Save your drawing as a PNG file.

## How to use
1. Open `index.html` in any browser.
2. Pick a color and start drawing.
3. Click "Download" to save your work.
